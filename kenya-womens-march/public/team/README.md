# Team Photos

Place team member profile photos in this directory.

## Naming Convention

Photos should be named using the team member's full name in lowercase with spaces replaced by hyphens, followed by `.jpeg`:

- `esther-mwakali.jpeg`
- `millicent-awino.jpeg`
- `michelle-kabucho.jpeg`
- `terry-ochola.jpeg`
- `sophie-ogutu.jpeg`
- `lydia-dola.jpeg`
- `beatrice-kamau.jpeg`
- `regina-mutiru.jpeg`
- `comfort-achieng.jpeg`
- `anne-wanjiru.jpeg`
- `anne-wanjiku.jpeg` (Secretariat)

## Image Requirements

- Format: JPG or PNG
- Recommended size: 400x400px or larger (square aspect ratio)
- The images will be automatically cropped to a circle/rounded square shape
- High quality, professional photos work best

## Fallback

If a photo is missing, a placeholder with the person's initials will be displayed automatically.
